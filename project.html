Red Fuse Digital Natives Website

August - November 2017

Upon completion of my internship as a marketing intern at Red Fuse, I utilized my skills in web development to create a website for the internship program. I created the website using CSS with Bootstrap, Javascript, and PHP. After completing the website, I created a means to allow the employees of Red Fuse to update the content of the website without any programming experience. Knowing they were familiar with the WordPress interface, I exported the website to WordPress as a template and created a Content Management System using the WordPress library for PHP.
